# 특허 침해 Risk 검토 시스템

R&D 특허팀용 특허 침해 분석 도구입니다.

## 🌐 사용

배포 URL: `https://[사용자명].github.io/[저장소명]/`

## ✨ 주요 기능

- **WIPS 엑셀 자동 발췌** — 대표도면, 서지사항, 한글 청구항을 자동으로 추출
- **2가지 보고서 포맷**
  - ① 독립항 요약 보고서 (전문 + 강조)
  - ② 독립항 구성요소 분해 보고서 (○/△/× 매칭)
- **AI 자동 분해 (Gemini)** — 청구항을 변리사 수준으로 자동 분해
- **도면 회전·확대** — 드래그 / 줌 / 회전, 구성요소 표 동시 표시
- **모니터링 필요 별도 관리** — 침해 Risk와 독립
- **다운로드** — 엑셀 (2시트) / HTML 보고서

## 📦 배포 방법 (GitHub Pages)

1. GitHub에서 새 저장소 생성 (예: `patent-review`)
2. `index.html` 파일을 저장소에 업로드
3. 저장소 **Settings** → **Pages** 메뉴
4. **Source**: `Deploy from a branch` 선택
5. **Branch**: `main` / `(root)` 선택 → Save
6. 1~2분 후 `https://[사용자명].github.io/patent-review/` 로 접속 가능

## 🔄 업데이트 반영

`index.html` 파일을 수정한 새 버전으로 교체하고 푸시하면, 1~2분 뒤 자동으로 최신 버전이 반영됩니다.

## 🔑 AI 자동 분해 사용

분해형 모드에서 「🔑 키」 클릭 → [Google AI Studio](https://aistudio.google.com/apikey)에서 발급받은 키를 입력하면 사용 가능합니다.

- 무료 한도: 분당 15회 / 일 1,500회
- API 키는 사용자 브라우저에만 저장됨 (외부로 전송되지 않음)

## ⚠️ 주의

- 시스템 코드는 공개되어 있지만, **사용자가 입력하는 특허 정보는 외부 서버에 저장되지 않습니다.** (브라우저 메모리에서만 처리)
- Gemini AI 분해 기능을 사용할 경우, 청구항 텍스트가 Google Gemini API로 전송됩니다. 영업비밀 청구항 사용 시 회사 정책을 확인하세요.
